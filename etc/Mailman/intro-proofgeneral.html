Proof General is a generic interface for proof assistants, currently based on Emacs. This mailing list is for the announcement of new versions of Proof General and discussions about using Proof General. Posting is only available to list members. If you have a problem with using or configuring Proof General, please contact da+pg-support@inf.ed.ac.uk instead of posting to this list. People who are interested in the development of Proof General should join the Proof General developers list.
